# 🧘 Irai Yoga – Web App

A modern, mobile-first wellness application built with pure HTML + Tailwind CSS.

## Pages

| File | Description |
|------|-------------|
| `index.html` | Home page – services overview, popular sessions |
| `listing.html` | Available sessions – therapist cards with booking |
| `booking.html` | *(coming soon)* Booking confirmation page |

## How to Run

Simply open `index.html` in any browser — no build step or server required.

## Tech Stack
- Pure HTML5
- Tailwind CSS (CDN)
- Google Fonts: Cormorant Garamond + DM Sans

## Navigation Flow
index.html → listing.html → booking.html
